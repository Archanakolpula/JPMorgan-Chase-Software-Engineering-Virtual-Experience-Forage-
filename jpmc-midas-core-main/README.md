# JPMorgan Chase Software Engineering Virtual Experience (Forage)

## 📌 About

This project is part of the JPMorgan Chase & Co. Software Engineering Virtual Internship on Forage.

## 🚀 Features

* Kafka-based transaction processing
* H2 database integration
* Transaction validation system
* Incentive API integration
* REST API communication

## 🛠 Tech Stack

* Java
* Spring Boot
* Apache Kafka
* H2 Database
* REST APIs

## 📚 What I Learned

* Backend system design
* API integration using RestTemplate
* Debugging using breakpoints
* Database relationships (JPA)
* Handling real-world transaction systems

## 📂 Project Structure

* `component` → Kafka listeners & processing
* `entity` → Database models
* `repository` → Database access
* `foundation` → Core classes
